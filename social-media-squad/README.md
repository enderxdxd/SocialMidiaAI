# Social Media Squad

Social media content creation squad — 6 experts + 4 specialists + 1 orchestrator.

## Quick Start

```
@social-chief           # Activate the orchestrator
*diagnose               # Triage your content request
*create-post            # Create a single post
*create-carousel        # Create a carousel
*setup-identity         # Set up visual identity for social media
*batch-content          # Create multiple posts at once
```
