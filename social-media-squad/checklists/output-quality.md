# Social Media Squad — Output Quality Checklist

## Visual Quality (Score 1-10)
- [ ] Brand colors used correctly (60-30-10 rule)
- [ ] Typography legible at mobile phone size
- [ ] Visual hierarchy clear — eye knows where to go first
- [ ] Composition follows platform best practices
- [ ] Image quality high enough for platform (min 1080px)
- [ ] Text-to-image ratio appropriate (Instagram < 20% text)

## Brand Consistency (Score 1-10)
- [ ] Logo present and correctly placed
- [ ] Colors match brand palette (exact hex values)
- [ ] Typography matches brand system
- [ ] Tone and mood align with brand personality
- [ ] Consistent with previous posts in the series

## Platform Optimization (Score 1-10)
- [ ] Correct dimensions for target platform
- [ ] Safe zones respected (stories, reels)
- [ ] Content format optimized (carousel for education, single for impact)
- [ ] Mobile-first design (tested at actual phone dimensions)

## Caption Quality (Score 1-10)
- [ ] Hook grabs attention in first 125 characters
- [ ] Body delivers value or tells a story
- [ ] Clear CTA aligned with content goal
- [ ] Hashtags relevant and strategic (5-15 for IG)
- [ ] Platform-native tone (not corporate speak)

## AI Image Prompt Quality (Score 1-10)
- [ ] All 7 layers present (subject, style, mood, lighting, composition, color, technical)
- [ ] Brand hex colors injected
- [ ] Text-safe zones specified
- [ ] Negative prompts include off-brand elements
- [ ] Platform-optimized (Flux for photo, MJ for artistic)

## Minimum Score: 7.0 average across all categories
## Below 7.0 → revision required before delivery
